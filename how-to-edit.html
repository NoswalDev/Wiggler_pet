How to edit (for dummies)(you)

You want to import a guide? Poor soul.
Anyway, here's how:

Step 1

Get guide. Got guide? hooray!
For readability, i strongly recommend downloading VS-Code (here).
Opening HTML files in it will highlight code, making it much more readable.

this is what this very website looks in it!
(it's a gif so the compression is weird, dont mind that)
Most of your work from here will be Copy-Pasting. The general Structure of Weapon-related guides is done, you just need to paste the words in the labelled spots and make sure the Images you want to use have the EXACT NAME as found within the html.
If the image reference says "swaxe1" for example, name it exactly that. Don't get fancy with capitalization, as this whole thing is extremely case sensitive. Don't add spaces either.
Example of how that's supposed to look:


Step 2

You've successfully copypasted your entire guide? Awesome! Save it as .html file, open your Browser and open that file to see if you've done it successfully and everything works as intended. If it does not, you either deleted something you weren't supposed to or added some < or > somewhere.
Your best bet is to not use the < and > signs at all.
For Weaponspecific guides, you'll also want to add the controls of the combos you talked about. This is a little bit finnicky, but you can find the copy-paste string at the top of the page.
